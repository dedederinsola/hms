  1. mysqli_query(): Used to execute a MySQL query.
php
$sql = "SELECT * FROM users";

if (mysqli_num_rows($result) > 0) {
    while ($row = mysqli_fetch_assoc($result)) {
        echo "Name: " . $row["name"] . ", Email: " . $row["email"];
    }
} else {
    echo "No results found.";
}
  1. mysqli_fetch_assoc(): Used to fetch a row from a result set as an associative array.
php
while ($row = mysqli_fetch_assoc($result)) {
    echo "Name: " . $row["name"] . ", Email: " . $row["email"];
}
  1. mysqli_close(): Used to close the connection to a MySQL database.
php

  1. mysqli_real_escape_string(): Used to escape special characters in a string for use in a MySQL query.
php
$name = mysqli_real_escape_string($conn, $name);
  1. mysqlnd_ms_query_is_select(): A function provided by the MySQL Native Driver (mysqlnd) extension to check if a query is a SELECT statement.

    // Code to handle SELECT query
}

Please note that some of these keywords are specific to the mysqli extension, which is commonly used for MySQL database interactions in PHP.
